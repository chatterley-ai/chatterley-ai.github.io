Downloads

Production installers are published via GitHub Releases. Use the links below once a release is created.

macOS

  • Download: Releases
  • Build locally: 
    cd frontend/chatterley
npm install
npm run dist:mac

Windows

  • Download: Releases
  • Build locally: 
    cd frontend\chatterley
npm install
npm run dist:win

Linux

  • Download: Releases
  • Build locally: 
    cd frontend/chatterley
npm install
npm run dist:linux

Backend (Docker)

cd backend/oumi
docker build -t chatterley-oumi .
docker run --rm -p 9000:9000 chatterley-oumi \
  python -m oumi.webchat.server --host 0.0.0.0 --port 9000
