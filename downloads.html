Installation Instructions
  1. Download the .dmg file
  2. Open the downloaded file
  3. Drag Chatterley to Applications folder
  4. Open Chatterley from Applications
  5. Important: The app is currently unsigned. If macOS prevents you from opening it, open Terminal and run: 
    xattr -cr /Applications/Chatterley.app
    Then try opening Chatterley again from Applications.

System Requirements

Minimum

  • RAM: 8 GB
  • Storage: 10 GB free space
  • CPU: Multi-core processor (4+ cores recommended)
  • GPU: Optional (enables faster inference for local models)

Recommended

  • RAM: 16 GB or more
  • Storage: 50 GB+ free space (for model storage)
  • GPU: NVIDIA GPU with 8+ GB VRAM (for local models)
  • Internet: Required for initial setup and API-based models
